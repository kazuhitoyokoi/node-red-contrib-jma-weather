node-red-contrib-jma-weather
================

気象庁の天気予報APIにアクセスするノードです。

**本ノードを利用するには、Node-RED 1.3以降の環境が必要です。**

## インストール

安定バージョンをインストールするには `メニュー - パレットの管理 - インストール` を使用してnode-red-contrib-jma-weatherを検索するか、Node-REDのユーザーディレクトリ（通常は`~/.node-red`）内で、次のコマンドを実行します。

    npm install node-red-contrib-jma-weather

## 利用方法

ノードプロパティ設定で「地域」を選択するのみで、指定した場所の天気を取得できます。
